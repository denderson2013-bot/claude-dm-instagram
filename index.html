1

Crie uma conta no ManyChat

Acesse manychat.com e crie uma conta gratuita. Conecte seu perfil do Instagram (precisa ser conta profissional/business).

2

Configure o trigger de comentários

No ManyChat, crie uma nova automação. Configure o trigger para detectar quando alguém comentar uma palavra-chave no seu post (ex: "EU QUERO"). Quando detectar, a automação envia uma DM automática pro seguidor.

3

Crie uma conta na Anthropic

Acesse console.anthropic.com e crie uma conta. Gere uma API Key na seção de API Keys. Essa chave conecta o Claude ao ManyChat.

4

Conecte o Claude ao ManyChat

No ManyChat, vá em Settings → Integrations → Claude (Anthropic). Cole sua API Key. Agora o Claude é o "cérebro" das suas automações.

5

Configure o fluxo de conversa

Crie o fluxo: DM automática de boas-vindas → Claude assume a conversa → responde dúvidas → qualifica o lead (pergunta o que precisa, qual o momento) → manda link de agendamento ou checkout. Escreva o tom de voz e as regras no prompt do Claude.

6

Teste e ative

Mande alguém comentar no seu post pra testar. Verifique se a DM chega, se o Claude responde naturalmente, se o link é enviado. Ajuste o tom de voz até ficar perfeito. Ative a automação.

💡 Dica: O segredo é o prompt

Escreva exatamente como você quer que a IA responda: tom de voz, o que pode e não pode falar, quais links enviar, quando escalar pra um humano. Quanto mais detalhado o prompt, mais natural a conversa.
